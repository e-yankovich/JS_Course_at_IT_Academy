# JavaScript Course at IT Academy

This repository contains coursework completed during a JavaScript course at IT Academy (Minsk, Belarus).

It includes practical assignments focused on JavaScript fundamentals, dynamic forms, validation, object-based data handling, and a final project built with HTML, CSS and JavaScript.

## Technologies Used

* HTML
* CSS
* JavaScript

## Final Project
### Blocks Builder

An interactive browser-based game where the player piles moving blocks onto each other to score points.
The project was built with HTML, CSS and JavaScript and includes dynamic rendering, user interaction handling and game logic implementation.

* [Open Project](https://e-yankovich.github.io/JS_Course_at_IT_Academy/Final%20project/Blocks_Builder.html)
